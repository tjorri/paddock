Detailed documentation is landing as v0.X stabilizes. The platform's internals
and UX are still being refactored toward v1.0; until they settle, the canonical
references live in the repo:

- [README on GitHub](https://github.com/tjorri/paddock#readme) — install,
  quickstart, repo layout.
- [ADRs](https://github.com/tjorri/paddock/tree/main/docs/adr) — architecture
  decision records.
- [Specs](https://github.com/tjorri/paddock/tree/main/docs/specs) — implementation
  specs for major milestones.
- [Cookbooks](https://github.com/tjorri/paddock/tree/main/docs/cookbooks) —
  runnable recipes for common provider, secret, and policy patterns.
- [Threat model](https://github.com/tjorri/paddock/blob/main/docs/security/threat-model.md)
  — the security boundary, in detail.

The pages in this section are deliberately scaffolded: stubs reserve URLs and
let the navigation settle now, so links shared during the skeleton phase don't
break when real content lands.

The one section that has real content right now is **[Concepts](/paddock/docs/concepts/)** —
the vocabulary primer. The vocabulary is stable even while internals refactor,
so that content is the safest to author today.
