# Superpowers

Outputs of the brainstorming and writing-plans skills (part of the
superpowers plugin) used during development. Two subdirectories matching
the skill conventions:

- [`specs/`](specs/) — design specs from the brainstorming skill.
  Filename pattern `YYYY-MM-DD-<topic>-design.md`.
- [`plans/`](plans/) — implementation plans from the writing-plans skill.
  Filename pattern `YYYY-MM-DD-<feature-name>.md`.

These are working documents, not user-facing reference material. They
remain useful as a record of how each feature was scoped and built.
