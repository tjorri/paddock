# Getting started

For evaluators and new operators. Pages here walk you through installing
Paddock and running your first harness.

Pages:

- [`quickstart.md`](quickstart.md) — kind cluster, both reference
  harnesses, `BrokerPolicy` setup, observability. Ten minutes.

Once you have Paddock running, [`../guides/`](../guides/) covers
task-oriented recipes (provider setup, delivery modes, allowlist
bootstrap), and [`../operations/`](../operations/) covers day-2 work.
