> Run AI agent harnesses as first-class Kubernetes workloads, with the safety rails built in.

Paddock is an open-source, Kubernetes-native platform for running headless AI agent harnesses ‚Äî Claude Code, Codex CLI, OpenCode, Pi, or anything else you can put in a container ‚Äî as templated, sandboxed, observable batch workloads. v0.3 adds a capability-scoped **broker** for credential issuance and a per-run **egress proxy** that MITMs TLS so the agent never sees upstream API keys.

> **Status:** pre-1.0. Expect breaking changes between minor versions until v1.0; pin to a tagged release for stability.

**Start at [`docs/`](docs/)** ‚Äî audience-routed entry point for evaluators, operators, harness authors, security reviewers, and contributors.

For deeper internal reading: [`VISION.md`](VISION.md) (product north star), [`docs/internal/specs/`](docs/internal/specs/) (implementation specs), [`docs/contributing/adr/`](docs/contributing/adr/) (architecture decisions).

## What's in the box

Paddock is built around five CRDs (`HarnessTemplate` / `ClusterHarnessTemplate`, `HarnessRun`, `Workspace`, `BrokerPolicy`, `AuditEvent`), a control plane (controller + admission webhooks + capability-scoped broker), and per-run sidecars (egress proxy + adapter + collector + a transparent-mode iptables-init). See [`docs/concepts/components.md`](docs/concepts/components.md) for the full inventory.

Reference harnesses: `paddock-echo` (deterministic CI fixture) and Claude Code (real-agent demo).

## Quickstart

A complete kind-cluster walkthrough ‚Äî local cluster, both reference harnesses (`paddock-echo` and Claude Code), `BrokerPolicy` setup, and the observability surface ‚Äî lives in [`docs/getting-started/quickstart.md`](docs/getting-started/quickstart.md). Runs end-to-end in about ten minutes.

Prerequisites: Go 1.26+, Docker, `kubectl`, [Kind](https://kind.sigs.k8s.io/) 0.25+, Kubernetes 1.29+ on the target cluster.

## Tests

- `make test` ‚Äî unit + envtest suites. Podspec goldens, reconciler behaviour, webhook admission, CLI plumbing, event/ring/tailer logic, provider + proxy correctness. Fast.
- `make test-e2e` ‚Äî Kind cluster + echo pipeline end-to-end. Slow; the load-bearing smoke test.
- `make lint` ‚Äî golangci-lint; config at `.golangci.yml` is deliberately loose on canonical Go idioms.
